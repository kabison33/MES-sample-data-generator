RouteOrg.xlsx
-------------
Defines all products made in the shopfloor and what operations they should undergo and which manufacturing cells/locations they should go to.

sn.xlsx
---
A counter for serial number for each part number

order.xlsx
---
A counter for order number for each part number

generator.py
----
takes in the files above and produce the sample output in the output folder

disc_generator.py
----
takes in the files and produce sample product defect data into the output folder
